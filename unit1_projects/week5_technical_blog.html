Ruby Classes 25 May 2014

A class is a template for creating object instances. Objects are created from that class type. Things an object knows about itself are called instance variables. They represent an object's state, (for example a Persons SSN number, or date of birth. ) and can have unique values for each object of that type.

Class Person # class Person
End

James = new.Person # object of class Person with the handle James

Similar to a constructor in Java, Ruby has a method named initialize that creates a new object and executes it. We use initialize to assign the parameter we passed into it to an instance variable.

Class Person
def initialize(name, age,)
@name = name
@age = age
puts "constructor called with two arguments."
end
end

Ruby like Java does not allow instance variables to be accessed outside of methods for that object. Ruby accessor methods are optional. Ruby has a group of attribute functions that can very easily create the necessary getters and setters. Attr-accessor creates the getter and setter methods for the specific attribute.

Attr_accessor :name

Is a shortcut for:

Class Person
def name(name) #setter function
@name = name
end

def name # getter function
@name
end
end

Using attr_accessor we saved ourselves the time of writing 2 methods. Instead we wrote 1 line of code.
