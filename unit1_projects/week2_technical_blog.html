CSS Properties: Margin, Border, and Padding 6 May 2014

Overview

All html block-level elements have five spacing properties: height, width, margin, border, padding. When using these properties, we are primarily working with < div > tags. < div > tags allow you to wrap, style, and position text in your webpage.

Border

Border properties allow you to specify the style and color of and elements border. ex border: 1px solid black;

Padding

Padding clear an area inside an element inside the border. Padding have 4 properties that can be changed independently top, bottom, left, and right. ex padding-left: 30px;

Margin

Margins clear an area around an element outside the border. Margins have 4 properties that can be changed independently top, bottom, left, and right. ex margin-top: 30px;
