Ruby Method: group_by 18 May 2014

This weeks blog will be about the ruby Enumerable method group_by. group_by groups the collection by result of the block statement and returns a hash where the keys are the evaluated result from the block and the values are arrays of elements in the collection that correspond to the key. If no block is given an enumerator is returned
The command: (1..6).group_by{ |grp| grp/2.0 == 2.0 }
Returns: {false=>[1, 2, 3, 5, 6], true=>[4]}
