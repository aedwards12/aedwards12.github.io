JavaScript in Web development 8 May 2014

JavaScript is mans best friend to a browser. The purpose of JavaScript is to improve your visitorâ€™s experience. JavaScript provides instant feedback. For example if a visitor is filling out a form with the use of JavaScript the input fields can be validated immediately with a go or no-go response versus pressing the submit button and waiting for the browser to refresh in hopes that your input was accepted. Another good example for using JavaScript would be for image galleries. Without the use of JavaScript each time the visitor requested a new image to view the browser would have to refresh to new page with that image. Instead with the use of JavaScript the image would populate on the page without refreshing the browser.
